# ASCII-Art

This program converts your picture to ASCII-art .txt file.

## Arguments:

### Argument 1:

--help, -h: help page

{$pic_path}: path to the picture

### Argument 2:

{$txt_path}: path to the output in .txt extension

## Example

<python main.py "C:\yp\our_picture.png" "C:\ascii.txt"
